## 本地运行

- 安装 Sphinx

  ```
    yum -y install git make python3 python3-pip
    pip3 install sphinx
    pip3 install sphinx-autobuild
    pip3 install sphinx_rtd_theme
    pip3 install recommonmark
    pip3 install sphinx_markdown_tables
  ```

- 运行

  ```
  sphinx-autobuild docs build/html
  ```
